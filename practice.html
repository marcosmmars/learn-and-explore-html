History in Shortcut for HTML

The first publicly available description of HTML was a document called "HTML Tags",[6] first mentioned on the Internet by Tim Berners-Lee in late 1991.[7][8] It describes 18 elements comprising the initial, relatively simple design of HTML. Except for the hyperlink tag, these were strongly influenced by CERN SGML, an in-house Standard Generalized Markup Language (SGML)-based documentation format at CERN. Eleven of these elements still exist in HTML 4.[9]


HTML is a markup language that web browsers use to interpret and compose text, images, and other material into visible or audible web pages. Default characteristics for every item of HTML markup are defined in the browser, and these characteristics can be altered or enhanced by the web page designer's additional use of CSS. Many of the text elements are mentioned in the 1988 ISO technical report TR 9537 Techniques for using SGML, which describes the features of early text formatting languages such as that used by the RUNOFF command developed in the early 1960s for the CTSS (Compatible Time-Sharing System) operating system. These formatting commands were derived from the commands used by typesetters to manually format documents. However, the SGML concept of generalized markup is based on elements (nested annotated ranges with attributes) rather than merely print effects, with separate structure and markup. HTML has been progressively moved in this direction with CSS.

HTML Elements I have learned

Paragraph
"P" will be the block level HTML tag for writing something as a block
Bold
"b" will be the inline HTML tag for making text bold
Strong
"strong" will be the inline HTML tag for making text bold and more important
Italic
"i" will be the inline HTML tag for making text italic
Emphasis
"em" will be the inline HTML tag for emphasizing text
Heading Tag
"h1" to "h6" will be the block level HTML tags for defining HTML headings
Head Tag
"head" will be the block level HTML tag for defining the head section of the document
Title
"title" will be the inline HTML tag for defining the title of the document
Line Break
"br" will be the inline HTML tag for inserting a line break
Comments
"!--" will be the block level HTML tag for defining comments in the document
Unordered List
"ul" will be the block level HTML tag for defining an unordered list
Ordered List
"ol" will be the block level HTML tag for defining an ordered list
List Item
"li" will be the block level HTML tag for defining a list item
Data Title
"dt" will be the block level HTML tag for defining a data title
Data List
"dl" will be the block level HTML tag for defining a data list
Data Description
"dd" will be the block level HTML tag for defining a data description
Anchor
"a" will be the inline HTML tag for defining a hyperlink
Image
"img" will be the inline HTML tag for defining an image
Form
"form" will be the block level HTML tag for defining a form
Input
"input" will be the inline HTML tag for defining an input field
Button
"button" will be the inline HTML tag for defining a button
Division
"div" will be the block level HTML tag for defining a division or section
Span
"span" will be the inline HTML tag for defining a span of text
Footer
"footer" will be the block level HTML tag for defining a footer section
Small
"small" will be the inline HTML tag for making text smaller
Label
